# csp.cr
Work-in-progress transliteration of of [libcsp](https://github.com/shiyanhui/libcsp).

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     csp:
       github: dscottboggs/csp.cr
   ```

2. Run `shards install`
